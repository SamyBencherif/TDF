TDF Language and Development Manual

Introduction

This document contains comprehensive instructions regarding use, development, and implementation of the Test Descriptor Format. This document is required for anyone tasked with developing test and abort sequences, proving TDF code, making use of TDF libraries, or developing mission critical peripheral technologies. Rule of thumb: if your project description includes TDF, this document is for you.
